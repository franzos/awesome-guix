# Contributing

Found a mistake, or want to add something guix-related? Open a PR.

Please follow the style of previous commits (check `git log`).

## Unmaintained entries

Entries with no updates for 6+ years are marked with a trailing 💤, e.g.:

```
- [some/guix-config](https://example.com/some/guix-config) 💤
```

The marker is a heads-up, not a removal — old configs in particular often stay useful as references. Entries that 404 or are archived/moved should be removed instead.
